# 7 整数反转
## Solution
``` python
class Solution:
    def reverse(self, x: int) -> int:
        if x >0 and x<2**31 -1:
            x = str(x)
            x = int(x[::-1])
            if x < 2**31 -1:
                return x
            else:
                return 0
        elif x<0 and x>1-2**31:
            x = str(x)
            x = int(x[1:][::-1])
            if x < 2**31 - 1:
                return -1*x
            else:
                return 0
        else:
            return 0
```
